# Category Icons - User Guide

## 🎯 Quick Start

Categories now support icons just like technologies! Upload icons from the WordPress Media Library.

---

## 📝 How to Add Icon to Category

### Step 1: Go to Categories Page
1. WordPress Admin → **Portfolio Showcase**
2. Click **Categories**

### Step 2: Add or Edit Category
- **New Category:** Fill in the "Category Name" field
- **Existing Category:** Click **Edit** next to the category

### Step 3: Upload Icon
1. Scroll to **"Category Icon"** section
2. Click **"Upload Icon"** button
3. WordPress Media Library opens

### Step 4: Select Icon
**Option A - Select Existing:**
- Click on any existing image in your media library
- Click **"Select"** button

**Option B - Upload New:**
- Click **"Upload files"** tab in media library
- Click **"Select Files"**
- Choose icon file from your computer
- Click **"Select"** button

### Step 5: Save
- Icon preview appears instantly
- Button changes to **"Change Icon"**
- **"Remove Icon"** button appears
- Click **"Add Category"** or **"Update Category"**

**Done!** Your category now has an icon. ✅

---

## 🔄 Managing Category Icons

### Change Icon
1. Edit the category
2. Click **"Change Icon"**
3. Select different image
4. Click **"Update Category"**

### Remove Icon
1. Edit the category
2. Click **"Remove Icon"**
3. Preview clears
4. Click **"Update Category"**

### Reuse Same Icon
1. Multiple categories can share the same icon
2. Simply select the same icon from media library
3. Saves space and keeps library organized

## 📋 Supported File Types

✅ **Recommended:**
- **SVG** - Best for icons (scalable, small file size)
- **PNG** - Good for icons with transparency
- **WEBP** - Modern, efficient format

✅ **Also Supported:**
- **JPG/JPEG** - Works but not ideal for icons
- **GIF** - Works but limited

💡 **Best Choice:** Use SVG files for perfect quality at any size!

---

## 📏 Icon Requirements

### File Size
- **Maximum:** 2MB
- **Recommended:** Under 50KB
- **Optimal:** 10-20KB

### Dimensions
- Icons display at **22x22 pixels** in admin table
- Can be used larger on frontend
- SVG scales perfectly to any size

### Quality Tips
- ✅ Use transparent background
- ✅ Simple, clear design
- ✅ Monochrome or 2-3 colors
- ✅ Square aspect ratio works best

## ❓ Frequently Asked Questions

### Can I use the same icon for multiple categories?
**Yes!** Select the same icon from media library for different categories. This is efficient and recommended.

### Do I need an icon for every category?
**No!** Icons are optional. Categories work perfectly without icons.

### What happens to the icon if I delete a category?
The icon remains in your media library. It's not deleted automatically, so you can reuse it.

### Can I change the icon later?
**Yes!** Edit the category anytime and change or remove the icon.

### Do icons appear on the frontend?
Icons are available in project data. Whether they display depends on your theme/frontend template configuration.

### What's the best icon format?
**SVG** - It scales perfectly, has small file size, and looks sharp at any size.

### Can I upload multiple icons at once?
Not to categories directly, but you can:
1. Upload all icons to media library first
2. Then assign them to categories one by one

### Do category icons work like technology icons?
**Yes!** They work exactly the same way with the same features.

### Are icons included in export/import?
**Yes!** When you export data, category icons are included and will be imported automatically.

## 🛠️ Troubleshooting

### "Upload Icon" button doesn't work
**Solutions:**
- Clear browser cache (Ctrl+F5)
- Try different browser
- Check JavaScript console (F12) for errors
- Refresh page and try again

### Can't upload SVG files
**Why:** WordPress blocks SVG uploads by default for security

**Solutions:**
- Install "Safe SVG" plugin
- Contact your developer to enable SVG support
- Use PNG format instead

### Icon doesn't appear after uploading
**Solutions:**
- Make sure you clicked "Select" in media library
- Check that preview shows before saving
- Try selecting the icon again
- Verify image file isn't corrupted

### Icon appears distorted
**Solutions:**
- Use square-shaped icons (1:1 aspect ratio)
- Use SVG format for best results
- Check original icon dimensions
- Try re-uploading at correct size

### Icon preview is blank
**Solutions:**
- Check that image actually uploaded to media library
- Try selecting a different image
- Clear browser cache
- Check file permissions

## ✨ Pro Tips

### Organization
1. **Naming Convention:** Use prefix like `category-ecommerce.svg`
2. **Folders:** Use media library folders plugin to organize
3. **Tags:** Add tags to icons in media library
4. **Descriptions:** Add notes in media library for future reference

### Efficiency
1. **Upload All First:** Upload all icons to media library at once
2. **Batch Assign:** Then quickly assign to categories
3. **Reuse:** Select same icon for similar categories
4. **Backup:** Export data regularly to backup icon assignments

### Design Consistency
1. **Same Set:** Use icons from the same icon pack
2. **Same Style:** Line icons OR solid icons, not mixed
3. **Same Colors:** Keep consistent color scheme
4. **Same Size:** Use icons with similar visual weight

### Performance
1. **Optimize Files:** Use tools like SVGOMG or TinyPNG
2. **Small Files:** Keep icons under 20KB when possible
3. **SVG First:** Prefer SVG over PNG for smaller files
4. **Lazy Load:** If showing many icons on frontend, consider lazy loading
